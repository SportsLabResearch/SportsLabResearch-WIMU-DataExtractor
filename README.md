# SportsLabResearch-WIMU-DataExtractor

High-performance open-source extractor for **WIMU®** `.QUL` and `.QUI` files.

## Features

- Native support for `.QUL` files
- Native support for `.QUI` files
- Automatic sensor detection
- Native Excel (`.xlsx`) export
- No pandas dependency
- No openpyxl dependency
- High-performance processing
- Large-file support

## Supported Sensors

- GPS
- Accelerometer
- Gyroscope
- Magnetometer
- Attitude
- Heart Rate
- Barometer
- Battery
- Status
