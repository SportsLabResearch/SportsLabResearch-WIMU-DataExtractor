# Getting Started

Get started with SportsLabResearch-WIMU-DataExtractor in a few minutes.

## Overview

SportsLabResearch-WIMU-DataExtractor provides a standardized workflow for extracting, validating, and reporting WIMU® performance data.

## Workflow

1. Select one or more WIMU® datasets.
2. Extract the available variables.
3. Standardize the dataset structure.
4. Validate the processed data.
5. Generate the reports.
6. Review the results.

## Requirements

| Component | Requirement |
|----------|-------------|
| Operating System | Windows 10/11 |
| Python | 3.11 or later |
| Microsoft Excel | Optional |
