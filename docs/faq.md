# Frequently Asked Questions



Answers to the most common questions about SportsLabResearch-WIMU-DataExtractor.



## Which file formats are supported?



The software supports WIMU® datasets exported in Microsoft Excel (`.xlsx`) and Comma-Separated Values (`.csv`) formats.



## Can multiple files be processed?



Yes. The application supports single-file, multi-file, and batch processing.



## Is validation mandatory?



No. Validation is optional and is recommended when reference datasets are available.



## Which reports are generated?



Depending on the selected workflow, the software can generate:



- Excel workbooks.

- Word technical reports.

- Validation reports.

- Processing logs.



## Where can I report an issue?



Issues and feature requests can be submitted through the project's GitHub Issue Tracker.
