# Technical Report TR-003 — Variable Dictionary

Describe the variables extracted and standardized by SportsLabResearch-WIMU-DataExtractor.

## Purpose

Provide a reference for all variables generated during data extraction and standardization.

## Variable Categories

| Category | Description |
|----------|-------------|
| Participant Information | Player and participant data. |
| Session Information | Session metadata. |
| External Load | Performance and movement metrics. |
| Internal Load | Physiological response variables. |
| Positioning Data | GNSS positioning variables. |
| Derived Variables | Calculated metrics generated by the software. |

## Variable Definition

Each variable includes:

- Name
- Description
- Unit
- Data Type
- Source

## Standardization

Variable names and formats are standardized to ensure compatibility across datasets and software versions.

## Outputs

The standardized variable dictionary is included in the generated Excel datasets and technical reports.
