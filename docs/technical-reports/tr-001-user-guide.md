# Technical Report TR-001 — User Guide

Describe the standard use of SportsLabResearch-WIMU-DataExtractor.

## Purpose

Provide a concise reference for the complete workflow, from data selection to report generation.

## Workflow

1. Select the input datasets.
2. Choose the processing mode.
3. Extract and standardize the data.
4. Validate the processed variables.
5. Generate the reports.
6. Archive the results.

## Input Data

Supported formats:

- Microsoft Excel (.xlsx)
- Comma-Separated Values (.csv)

## Outputs

- Standardized Excel datasets.
- Word technical reports.
- Validation reports.
- Processing logs.

## Best Practices

- Preserve the original datasets.
- Process complete exports.
- Archive reports with the processed data.
- Record the software version.

## Limitations

- Windows is the recommended platform.
- Input files must follow the supported WIMU® export structure.
