# Validation

Scientific validation resources for SportsLabResearch-WIMU-DataExtractor.

## Scope

This section documents the comparison between the extraction script and SPRO reference outputs.

## Available Resources

- Validation overview.
- Validation protocol.
- Validation results.
- Validation report in PDF.

## Main Findings

- High overall agreement between tools.
- Excellent concordance for position, GPS speed, and acceleration variables.
- Isolated discrepancies in some gyroscope and magnetometer variables.
- Review of units, scaling, and reference frames is recommended for discrepant variables.
