# Menus

Navigate the application through a simple menu designed to support the complete processing workflow.

## Main Menu

The main menu provides access to the core functions of the application.

| Option | Description |
|---------|-------------|
| Process Single File | Process a single dataset. |
| Process Multiple Files | Process multiple datasets. |
| Batch Processing | Process all available datasets. |
| Validation | Compare results with the reference software. |
| Exit | Close the application. |

## Processing

Each option guides the user through file selection, processing, validation, and report generation.

## Recommendations

- Verify the selected input files.
- Process complete datasets whenever possible.
- Save results in a dedicated output folder.
- Use the latest stable release.
