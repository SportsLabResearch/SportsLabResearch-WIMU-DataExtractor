# Troubleshooting

Resolve the most common issues encountered during data processing and report generation.

## Overview

Most issues are related to unsupported input files, missing dependencies, or incorrect project configuration.

## Common Issues

| Issue | Solution |
|-------|----------|
| Input file cannot be opened | Verify that the file exists and is not being used by another application. |
| No data extracted | Confirm that the selected file is a supported WIMU® export. |
| Validation failed | Ensure that both comparison files belong to the same dataset. |
| Report generation failed | Verify that the output folder is writable and that report files are closed. |

## Before Requesting Support

Verify that:

- The latest software version is installed.
- All required dependencies are available.
- Input files are complete and valid.
- The issue can be reproduced.

## Support Information

Include the following information when reporting an issue:

- Software version.
- Operating system.
- Description of the problem.
- Steps to reproduce the issue.
- Error message, if available.
