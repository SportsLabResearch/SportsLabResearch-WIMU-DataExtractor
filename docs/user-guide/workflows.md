# Processing Workflows

Process WIMU® datasets using a standardized and reproducible workflow.

## Overview

The software follows a sequential workflow from data extraction to report generation, ensuring consistent processing across all datasets.

## Workflow

```text
Input Files
      │
      ▼
Data Extraction
      │
      ▼
Data Standardization
      │
      ▼
Quality Control
      │
      ▼
Validation
      │
      ▼
Report Generation
```

## Processing Steps

1. Select one or more WIMU® datasets.
2. Extract the available variables.
3. Standardize the dataset structure.
4. Perform automatic quality control.
5. Validate the results against reference data (optional).
6. Generate the output reports.

## Outputs

- Standardized Excel datasets.
- Word technical reports.
- Validation summaries.
- Processing logs.
