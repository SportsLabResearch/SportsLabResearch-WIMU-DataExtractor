# Reports

Generate standardized outputs for scientific analysis, validation, and documentation.

## Overview

The software generates reports that summarize the processed datasets, quality control results, and validation outcomes.

## Available Reports

| Report | Description |
|---------|-------------|
| Excel Workbook | Standardized datasets ready for analysis. |
| Word Report | Technical summary of the processing results. |
| Validation Report | Concordance analysis with the reference software. |
| Processing Log | Execution summary and diagnostic information. |

## Report Contents

Reports include:

- Project information.
- Processing summary.
- Processed datasets.
- Variable overview.
- Quality control results.
- Validation results.

## Recommendations

- Preserve all generated reports.
- Archive reports with the original datasets.
- Record the software version used.
- Retain reports as supporting material for reproducible research.
