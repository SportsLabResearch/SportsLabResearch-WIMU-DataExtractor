\# Contributing



Gracias por tu interés en contribuir a \*\*SportsLabResearch-WIMU-DataExtractor\*\*.



\## Objetivos del proyecto



\- Extraer datos de archivos WIMU `.QUL` y `.QUI`.

\- Mantener la máxima fidelidad respecto a los datos originales.

\- Priorizar rendimiento y estabilidad.

\- Evitar dependencias innecesarias.

\- Mantener compatibilidad con archivos de gran tamaño.



\## Normas



\- Código documentado.

\- Tipado (`type hints`) cuando sea posible.

\- Compatibilidad con Python 3.11 o superior.

\- Mantener el estilo del proyecto.

\- No introducir dependencias externas salvo justificación técnica.



\## Flujo de trabajo



1\. Crear una rama desde `dev`.

2\. Implementar la mejora.

3\. Verificar que no rompe funcionalidades existentes.

4\. Abrir un Pull Request con una descripción clara.



\## Reporte de errores



Incluye siempre:



\- Versión del programa.

\- Sistema operativo.

\- Archivo utilizado (`.QUL` o `.QUI`).

\- Mensaje completo del error.

\- Pasos para reproducirlo.
